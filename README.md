# 🎨 CSS Mini Projects Collection

This repository contains a collection of small CSS projects built to practice and improve front-end development skills. Each project focuses on a specific UI component or design concept.

---

## 📂 Projects Included

### 1. Glassmorphic Card

A modern card design using glassmorphism effect with blur and transparency.

### 2. Doodle Search Bar

A creative and fun search bar inspired by playful UI styles.

### 3. Custom Checkbox

Stylish and animated checkbox built purely with CSS.

### 4. Toggle Switch

A smooth and interactive toggle switch UI component.

### 5. Dropdown Menu

Responsive and clean dropdown navigation menu.

### 6. Profile Card

A simple user profile card with clean layout and styling.

### 7. Tooltip

Custom tooltip that appears on hover with smooth animation.

### 8. Navbar

A responsive navigation bar for modern websites.

### 9. Grid Cards

A card layout using CSS Grid for responsive design.

---

## 🚀 Technologies Used

* HTML5

* CSS Grid
* Animations & Transitions
